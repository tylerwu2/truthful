# Icon Creation Notes

For the Chrome Web Store submission, you need to create the following icons:

## Required Icons
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Design Guidelines
- Use a simple, recognizable design
- Primary color: Blue (#007bff) or similar
- Include a shield or checkmark to represent "verification" or "detection"
- Keep it clean and professional

## Tools
- Use Canva, Figma, or Adobe Illustrator
- Or online icon generators
- Make sure background is transparent

## Example Design
- A shield with a checkmark
- Or a magnifying glass over text
- Or a warning triangle with an eye

## Current Status
Manifest currently has icons commented out for development.
Uncomment and add actual PNG files before store submission.
