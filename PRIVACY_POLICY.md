# Privacy Policy for Truthful

## Data Collection and Usage

The Misinformation Detector Chrome extension does not collect, store, or transmit any personal information about users. All analysis is performed locally on the user's device.

## Information We Access

- **Webpage Content**: The extension reads the text content of web pages you visit to analyze for potential misinformation
- **Browser Storage**: Analysis results are temporarily cached in your browser's local storage (chrome.storage.local) for performance
- **Extension Settings**: Your chosen analysis provider and API keys (if provided) are stored locally

## Third-Party Services

When you choose to use AI-powered analysis:

- **Groq API**: Text is sent to Groq's servers for analysis using their Llama models
- **Google Gemini**: Text is sent to Google's Gemini API
- **Ollama**: Text is sent to your local Ollama instance (if running)

These services may have their own privacy policies. We recommend reviewing them before use.

## Data Retention

- Analysis results are cached for 1 hour to improve performance
- Cached data is stored locally on your device only
- No data is transmitted to our servers or third parties except for the chosen AI service

## Data Security

- API keys are stored securely in browser storage
- No data is logged or tracked
- All analysis happens in real-time and results are not stored permanently

## Contact

For privacy concerns, please check the extension's GitHub repository or contact the developer through Chrome Web Store.

## Changes to This Policy

This privacy policy may be updated. Check the extension's documentation for the latest version.
